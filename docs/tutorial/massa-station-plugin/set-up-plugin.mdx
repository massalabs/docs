Let's go through the steps to set up your plugin's environment.

## Install Go and Go dependencies

You can install Go [here](https://go.dev/doc/install).
Install Go swagger: `go install github.com/go-swagger/go-swagger/cmd/swagger@latest`


## Initialize the Go plugin

- Create and navigate to the folder my-plugin
- Run:

  ```bash
  go mod init my-plugin
  ```

### Create a Logo

- Create the folders `web/content` and add `hello_world.svg` in it.
- Add your custom logo or copy-paste [this one](https://github.com/massalabs/station-massa-hello-world/blob/main/web/content/hello_world.svg?short_path=1cd8e84):

### Add a manifest file

- Create `manifest.json` in the root of your project.
- Paste the following content, replacing the values with your own:

```json
{
  "author": "My Name",
  "name": "my-plugin",
  "description": "hello world plugin",
  "logo": "hello_world.svg",
  "home": "",
  "version": "0.0.6",
  "apispec": ""
}
```

You're now ready to start building your plugin!
